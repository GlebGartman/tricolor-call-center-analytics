## 🧠 SQL задачи
- Подсчет клиентов по типу оборудования с окончанием услуги в 3 квартале 2022 года  
- Определение последней неактивной услуги клиента

    current_service VARCHAR(50),
    status BOOLEAN,

### 📊 Решение задачи 1  
**Клиенты по типу оборудования (3 квартал 2022)**
```sql
SELECT
    type_of_equipment,
    COUNT(DISTINCT ap.id_client) AS clients_cnt
FROM active_periods AS ap
JOIN data_mart AS dm ON ap.id_client = dm.id_client
WHERE end_date BETWEEN '2022-07-01' AND '2022-09-30'
GROUP BY type_of_equipment
ORDER BY clients_cnt DESC;
```
---

### 📊 Решение задачи 2  
**Последняя неактивная услуга клиента**
```sql
WITH finish AS (
    SELECT
        id_client,
        service,
        end_date,
        ROW_NUMBER() OVER (PARTITION BY id_client ORDER BY end_date DESC) AS rnk
    FROM active_periods
    WHERE end_date < NOW()
)
SELECT
    id_client,
    service,
    end_date
FROM finish
WHERE rnk = 1;
```
