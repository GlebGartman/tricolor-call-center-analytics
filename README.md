<h3 align="center">Описание проекта</h3>
<p align="center">
Проект по продуктовой и операционной аналитике контакт центра Триколор, построение витрины и дашбордов в Power BI, расчет ключевых метрик и KPI, анализ операторов и тематики, SQL задачи по данным клиентов, прогноз подключений в Excel и оформление результатов в презентации
</p>

---

## 🎯 Цели проекта
- Построить витрину данных для анализа контакт центра  
- Рассчитать ключевые KPI  
- Проанализировать эффективность операторов  
- Определить проблемные и значимые тематики  
- Построить прогноз подключений  
- Подготовить выводы и рекомендации  

---

## 📊 KPI метрики
- Количество поступивших контактов  
- Количество отвеченных контактов  
- Доля неотвеченных контактов  
- Доля повторных контактов  
- Среднее время ожидания  
- Среднее время обработки (AHT)  
- Доля переведенных контактов  

---

## 📈 Аналитика
- Динамика KPI во времени  
- Анализ тематик:
  - наиболее длительные  
  - наиболее значимые  
- Анализ операторов:
  - лучший по AHT  
  - худший по AHT  

---

## 📉 Прогнозирование (Excel)

- Данные были предварительно нормализованы: устранено влияние акций и скидок с помощью линейной интерполяции и корректирующего коэффициента  
- Выделена трендовая составляющая с использованием линейной регрессии (ПРЕДСКАЗ.ЛИНЕЙН)  
- Рассчитаны сезонные коэффициенты как отношение значений месяца к среднему уровню  
- Применена мультипликативная модель временного ряда:  
  **Yt = dt × rt**  
- Итоговый прогноз получен путем умножения тренда на сезонные коэффициенты  
- Проведена валидация модели через анализ линии тренда и уравнения регрессии
